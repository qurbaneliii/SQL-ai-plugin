# Roadmap

## Current MVP Scope

- Standalone local web sidecar
- GitHub Pages safe frontend demo
- Backend-only OpenAI/Ollama/PostgreSQL access
- Deterministic SQL validation and read-only execution
- Schema-aware generation context
- Result preview and lightweight summarization

## Future Work

- Add persisted multi-session chat history and saved query notebooks
- Add richer schema relationship visualization
- Add optional `EXPLAIN (FORMAT JSON)` and plan analysis workflows
- Add explicit future write-mode with additional approvals and audit logging
- Add desktop wrapper packaging
- Add native SQL editor integrations where stable APIs exist
- Add authenticated backend deployment recipes for teams
